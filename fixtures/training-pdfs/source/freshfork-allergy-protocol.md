# Allergy Disclosure & Cross-Contamination Protocol

**Company:** FreshFork Restaurants | **Doc ID:** FOH-ALL-003 | **Version:** 2.0 | **Effective:** 2026-01-20 | **Owner:** Culinary & Safety

## Purpose & scope

All servers, hosts, and expo staff at FreshFork Restaurants must follow this protocol when a guest discloses a food allergy or asks about ingredients. Applies to dine-in, patio, and takeout with allergy flags in POS.

## Key terms

- **Big 9:** Milk, eggs, fish, shellfish, tree nuts, peanuts, wheat, soy, sesame.
- **Allergy ticket:** Kitchen copy printed in red with allergen code; expo verifies before pass.
- **Cross-contact:** Allergen transferred via shared fryer, grill, or utensils — not the same as cross-contamination in manufacturing.

## Standard procedure

1. When guest mentions allergy, pause order entry; notify manager if severity stated (anaphylaxis, EpiPen).
2. Read back allergen and all ordered items; never guess — check allergen matrix or ask chef.
3. Enter ALLERGY alert in POS; attach table number and server initials.
4. Confirm with kitchen that dedicated pan / clean gloves used; expo signs allergy ticket.
5. Deliver food yourself when possible; confirm with guest: "This was prepared for your [allergen] allergy."
6. Document incident if reaction suspected — manager completes IR-FOH same shift.

## Approved phrases

> "Thank you for telling me — I'll note that for the kitchen and double-check every ingredient."

> "Our fryer is shared — I want to flag that for your shellfish allergy before you decide on the fries."

> "I'll have the chef come out if you'd like to discuss how we handle tree nuts on this line."

> "Your plate was prepared separately with clean gloves — please let me know if anything looks wrong."

## Scenarios

**Scenario A — Peanut allergy, dessert question:** Guest orders entrée safely; asks if chocolate cake is safe. Server checks matrix — cake contains almond flour; offers fruit sorbet prepared on separate station.

**Scenario B — Gluten-free pasta request:** Guest celiac. Server confirms GF pasta boiled in separate water; warns about shared pasta water if kitchen busy — offers salad alternative if risk unacceptable.

**Scenario C — Child with dairy allergy:** Parents order kids' meal. Server flags allergy ticket, removes cheese from line, confirms butter not used on grill for that ticket only.

## Escalation matrix

| Situation | Action |
|-----------|--------|
| Guest reports reaction | Manager + 911 if severe; preserve plate per health dept guide |
| Kitchen cannot guarantee safety | Manager explains limitation; guest chooses to stay or leave |
| Guest demands guarantee in writing | Manager only — use approved liability language |
| Repeat allergy miss on same shift | Stop service; chef retraining same day |

## Do / Don't

**Do:** Write allergy on check legibly; carry allergy plates separately; wash hands after handling allergens.

**Don't:** Say "I'm pretty sure it's fine"; remove croutons and call salad gluten-free without checking dressing; hide allergy from kitchen to speed ticket.
